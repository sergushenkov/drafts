. Пример для ковариантностью подобрать легко, эта схема вполне понятна, естественно отражает иерархию наследования:

```py
class Primus:
    def burn(fuel)

class PropanePrimus(Primus):
    def burn(propan)

class GasPrimus(Primus):
    def burn(gas)
```
